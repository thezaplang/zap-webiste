---
title: declaring variables
description: how to create an variable in Zap
---

Creating variables in zap is as easy as in go, however it will be closer to rust than go.
variables have the following syntax:

```zap
let name: type = <expr>;
```

for example

```zap
fn main() -> i32{
    let a: i32 = 5;
    return 0;
}
```
